# Development workflow

## Build
```sh
make build
```

## Format and vet
```sh
make fmt
make vet
```

## Lint
```sh
make lint
```

## Tests
```sh
make test
```

## API docs
```sh
make docs
```

## Local docs preview (VitePress)
```sh
make startdocs
```

## Codegen and manifests
```sh
make generate
make manifests
```
